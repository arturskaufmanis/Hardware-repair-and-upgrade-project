Overview

Project Summary

This lab documents the complete lifecycle of an IT hardware repair and upgrade project — from fault diagnosis and structured research through to component sourcing, full chassis disassembly, keyboard replacement, RAM upgrade, reassembly, and systematic post-repair verification.

The device was purchased with a known keyboard fault. Rather than simply replacing the part, the project began with structured research: identifying the root cause of the fault, confirming the correct replacement part, assessing the platform's upgrade potential, and studying the disassembly procedure before touching the hardware. This approach reflects how hardware work should be conducted in a professional IT environment — methodically, with every decision grounded in research rather than assumption.

Fault Analysis

Diagnosis

On receipt of the device, a full inspection was carried out before any disassembly. The objective was to understand the actual failure mode — not assume it.

⌨️
Symptom

Three keys — R, T, and Y — were completely non-functional. All adjacent keys tested as operational.

🔍
Root Cause

Physical inspection revealed the previous owner had attempted a DIY repair. The aluminium stabiliser hinges on all three keycaps were sheared — consistent with improper keycap removal (lifted at an angle rather than straight up).

✅
Decision

Individual hinge repair was ruled out as non-viable. A full keyboard replacement was confirmed as the correct resolution. All other components were verified functional prior to proceeding.

Pre-Repair

Research Phase

Before ordering any parts or beginning disassembly, the following research was completed. This phase is a critical professional discipline — it ensures correct parts are sourced, no steps are improvised, and the repair is done once, correctly.

// Keyboard Replacement

  • ›Identified the correct HP spare part number for the EliteBook 845 G7 keyboard (UK layout)
  • ›Confirmed the keyboard is riveted to the palmrest — not screwed — requiring a heat-based separation method rather than standard screw removal
  • ›Reviewed the HP service manual disassembly sequence to identify the correct order of operations and flag fragile connectors and cables requiring care

// RAM Upgrade

  • ›Confirmed the 845 G7 has two SODIMM slots — with one occupied by a single 8 GB module, leaving the second slot available
  • ›Verified the platform supports dual-channel memory operation when both slots are populated with matched modules
  • ›Confirmed maximum supported capacity (32 GB) and DDR4 3200 MHz SO-DIMM as the correct specification
  • ›Sourced a matched 8 GB DDR4 3200 MHz module — enabling dual-channel operation for improved memory bandwidth, particularly beneficial for the AMD integrated GPU which draws from system memory

Methodology

Procedure

PHASE 01 Pre-Disassembly Preparation
  • Device disconnected from mains and visually inspected
  • Service manual reviewed; disassembly sequence confirmed
  • All tools and replacement parts prepared in advance
  • ESD precautions observed throughout
PHASE 02 Disassembly
  • Bottom panel removed using plastic picks and screwdriver
  • Battery connector disconnected immediately — standard practice to eliminate short-circuit risk before handling internal components
  • All components identified, inspected, and placed in disassembly order for controlled reassembly
  • Single 8 GB RAM module confirmed in slot 1; slot 2 empty
PHASE 03 Keyboard Replacement
  • Palmrest assembly removed from chassis
  • Heat gun applied to reverse of palmrest to soften factory rivet adhesive — controlled application to avoid thermal damage to PCB or ribbon cables
  • Damaged keyboard separated cleanly from palmrest
  • Replacement keyboard fitted and aligned
  • Glue gun applied at original rivet contact points — a deliberate technique choice creating a secure, reversible bond; future replacement possible by reheating, without palmrest damage
  • Keyboard ribbon cable and backlight connector reseated and verified
PHASE 04 RAM Upgrade
  • Second SODIMM slot located on motherboard
  • New 8 GB DDR4 3200 MHz module installed in empty slot
  • Both modules seated and latched; dual-channel configuration confirmed
PHASE 05 Reassembly
  • All components reinstalled in reverse disassembly sequence
  • All ribbon cables, antenna cables, and board connectors reseated
  • Battery reconnected; bottom panel refitted and all screws secured
